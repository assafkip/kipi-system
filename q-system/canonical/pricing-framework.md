## Pricing Tiers <!-- pin -->

(populated through debriefs and calibration)

### Tier: Managed local investigation OS setup for Chris / PI work
- **Target:** Boutique private investigator / investigation operator who wants to use Assaf's investigation workflow internally.
- **Price point:** $6,500 fixed setup, $1,500/month support, $150/hour for investigation work, custom workflows, or operational support.
- **Packaging:** Managed local deployment of the investigation system, configured for the client's case process. Includes validation of workflow and handoff. Does not include raw source ownership, Assaf's real case data, API keys, local settings, invoices, or broader Kipi/ASK/Catalyst system.
- **Proof points:** Price is based on replacement value and existing tool value, not only install hours. The system already exists and was refined through real investigation work. Value drivers are time saved, risk reduced, attribution discipline, source tracking, chain of custody, report workflow, and revenue enabled for client work and training.
- **Objection handling:** If asked why this is not just hourly setup, explain that the buyer is not paying for a blank build. They are paying to adapt an existing investigation operating system with guardrails, templates, and workflows already in place.
- **Source:** Assaf / Chris PI pricing discussion - 2026-06-30

## Budget Signals from Conversations

(captured from debrief pricing/packaging analysis)

### Chris / PI investigation system pricing signal
- Chris asked about getting Assaf's investigation tool working on his side.
- Assaf's position: do not sell the complete tool as a one-time handoff. Offer a managed internal deployment plus ongoing support.
- Hourly anchor: $150/hour, based on the range Assaf already named to Chris ($100-$150/hour) and the specialized nature of the work.
- Setup anchor: $6,500, framed as adapting an existing working investigation OS, not installing files.
- Support anchor: $1,500/month, roughly 10 hours/month of maintenance, workflow adjustments, and troubleshooting.
- Talk track: "The price is based on the value of the working system, not just the hours to install it."
